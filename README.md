## 📝 소개
- **SCSS**로 구현한 웹 페이지입니다.
- 스크롤로 토글되는 GNB 구현
- 토글되는 모달 구현
- Drawer(collapse)을 구현

## 🔖 특징
- 반응형 디자인
- Mixin으로 **중복되는 코드** 정리
- **컴포넌트 단위**로 디렉토리 관리
- 아이콘을 폰트화하여 **HTML 코드의 가독성**을 개선
